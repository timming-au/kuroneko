# kuroneko
Low poly world in threejs. All assets by me.<br>
Use keyboard controls to move around, or pan with your fingers on touchscreen devices.

## stack
* typescript
  * threejs
  * svelte
    * threlte
* tailwindcss
* blender
* midjourney
